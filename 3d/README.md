# The Idea Zoo: Glassmarket 3D

A dependency-free WebGL2 vertical slice of The Idea Zoo.

## Run locally

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000/3d/` when using the repository root, or open the served folder directly.

## Controls

- WASD / arrow keys: move or drive
- Mouse: camera
- E: interact / enter / exit / load cage
- Space: use selected containment tool
- 1–4: Praise, Money, Enemy, Silence
- Q: civic siren

The game includes desktop and mobile touch controls.
