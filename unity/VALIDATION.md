# Unity validation boundary

The Unity branch is validated in layers because this environment does not include a licensed Unity editor or a physical iPhone.

## Engine-independent domain contract

The .NET contract compiles the exact `IdeaZooDomain.cs` file and verifies:

- a complete intake → hatch → test → Molt → ruling path;
- duplicate evidence rejection;
- premature Molt rejection;
- unchanged Molt rejection;
- Molt cancellation recovery;
- evidence preservation through a Molt;
- strict token boundaries for `maintenance`/`AI` and `unpaid`/`paid`;
- zero-strength evidence not inflating the score;
- final ruling and three real-world actions.

## Runtime source contract

The source regression contract verifies:

- joystick reuse of the existing `RectTransform`;
- camera-touch reset when controls lock;
- modal and prompt layering;
- two-step ruling confirmation;
- evidence preservation;
- strict classification boundaries;
- corrupt archive quarantine and backup.

## Production and presentation structure contract

The structural audit verifies:

- Unity 6.5 project and package manifest;
- all gameplay, archive, scene and presentation files;
- all ten institution spaces;
- shared civic material surfaces and modular architecture kit;
- department-specific world dressing and environmental story language;
- six specialist names, tools and animated rigs;
- seven class silhouettes and eight appetite markings;
- hidden burden, evidence definition and guardrail presentation;
- authored-proportion protection;
- PlayableDirector story shots and optional Cinemachine bridge;
- civic ambience and story cues;
- mobile distance culling, device tiers and adaptive render scale;
- editor commands for prefab and review-scene baking;
- balanced braces, no unfinished production paths and no transport remnants.

The complete validation report is uploaded on every run, including failed runs.

## Not yet validated

A real Unity editor import, shader compilation, generated prefab bake, native player build and physical-device test still require Unity 6000.5.0f1. iOS additionally requires macOS, Xcode, signing and an actual iPhone.
